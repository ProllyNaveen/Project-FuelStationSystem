# ⛽ Fuel Station System

A C# WinForms desktop application developed as a coursework project for the Certificate in Software Engineering at NIBM. Manages fuel station operations including transactions, employee management and inventory management.

## Tech Stack
- C# (Visual Studio)
- SQL Server

## User Roles
- **Admin** — Full access to all features
- **Cashier** — Record transactions and manage daily operations

## Features
- Transaction tracking
- Employee management
- Inventory management

## 🚀 Quick Run
1. Go to the [Releases](https://github.com/ProllyNaveen/Project-FuelStationSystem/releases) tab
2. Download `FuelStationSystem_Setup.exe`
3. Run the installer
4. Open SSMS → Restore database using the file from the DATABASE folder
5. Make sure SQL Server is running
6. Launch from desktop shortcut ✅
